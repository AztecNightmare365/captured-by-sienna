Sign in

Paste your GitHub access token. It stays only in this browser (never uploaded, never in the code). You do this once per device.

One-time setup:
1. Go to github.com/settings/personal-access-tokens/new
2. Token name: "Sienna galleries". Expiration: 1 year.
3. Resource owner: AztecNightmare365. Repository access → Only select repositories → captured-by-sienna.
4. Permissions → Repository → Contents: Read and write.
5. Generate, copy the github_pat_... string, paste above.
